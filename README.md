## Due Date Calculator

### Setup
* git clone https://github.com/gabeszrozsa/due-date-calculator.git
* cd .\due-date-calculator\
* npm install

### Web App
* npm start

### CLI - node . YYYY-MM-DD HH:mm amount
* first param: date
* second param: time
* third param: turnaround time
* for example: node . 2017-10-27 14:41 5

### Unit tests
* npm test
